# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-01

### Added
- Initial release of RevisionOS
- Module-based document management and study flow generation
- Flashcard creation and spaced repetition review system
- AI-powered quiz generation with multiple difficulty levels
- AI tutor for interactive study assistance
- Knowledge graph visualization
- Curriculum planning with exam date targets
- Weakness mapping and adaptive session optimization
- Gamification with streaks, XP, and achievements
- Social leaderboard
- Dark/light theme support
- Document upload supporting PDF, text, slides, docs, audio, video, and images
- Notion and Google Drive integrations
- Anki and JSON export
